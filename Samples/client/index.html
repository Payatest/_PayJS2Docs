Getting Started

Before you get started, make sure any one of the server samples is also running, eg:

cd ../server/expressjs && npm install && npm start

You can use any of these fake card numbers:

4111111111111111 (Visa)
5454545454545454 (MasterCard)
6011000993026909 (Discover)
371449635392376  (American Express)

Or, for ACH transactions:

056008849      (Routing Number)
12345678901234 (Account Number)
Checking       (Account Type)
